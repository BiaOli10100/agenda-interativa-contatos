# Agenda Interativa de Contatos

## Descrição
Projeto desenvolvido para a disciplina de Programação Web I.
Consiste em uma agenda interativa que permite cadastrar, editar,
remover e visualizar contatos.

## Funcionalidades
- Cadastro de contatos (nome, telefone, email e grupo)
- Validação de telefone (mínimo 9 dígitos)
- Validação de email(tem que ter @ )
- Listagem dos contatos em tabela
- Edição e remoção de contatos
- Filtro por grupo(amigos,familia,trabalho e igreja)
- Campo de busca

## Tecnologias Utilizadas
- HTML5
- CSS3 (Grid Layout e Flexbox)
- JavaScript
